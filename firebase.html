TARGET AUDIENCE

The target audience is developers who want to use a custom domain for their Firebase-hosted web application. Connecting a custom domain is a one-time process and developers are not expected to be knowledgable about web hosting. Additionally, these users are developers who use the Firebase console and want to integrate in-app messaging into their applications. Users are expected to have an existing Firebase app, but they don't need prior experience with implementing messaging.
